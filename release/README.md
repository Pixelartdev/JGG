# Release

Hier stehen immer die aktuellsten Versionen der Apps zum download bereit. 
Für alle offenen Plattformen (Android, Windows, Mac OS,...) können die Apps direkt installiert werden.
Bei Plattformen wie iOS kann man zwar die App runterladen, braucht aber besondere Rechte um diese zu installieren (Entwickler Gerät, Jailbreak, ...).
